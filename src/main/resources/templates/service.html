일부 서비스 이용만 가능합니다.


게시판에 등록된 글만 확인하실 수 있습니다.


더 많은 서비스 이용을 원하시면 회원가입 해주시길 바랍니다.




개인회원 가입은 크게 두가지가 있습니다.


메인 페이지 우측 [회원가입] 을 통해 가능합니다.



[petdical bowwow] 계정 회원가입


1. [회원가입] 클릭




2. [개인회원 가입] 클릭 > 개인정보 입력



3. 회원가입 완료






[소셜] 회원가입


1. 카카오 로그인 [버튼] 클릭



2. 카카오 아이디 로그인 또는 QR코드 로그인 하기



3. 전체 동의 후 [동의하고 계속하기] 클릭




☞회원가입 바로가기(클릭)


로그인 영역의 아이디/비밀번호 찾기'를 통해 알 수 있습니다.




아이디 찾기 : 등록된 [이메일] , [연락처] 입력 → [아이디 찾기] 버튼 클릭




[비밀번호 찾기] : 등록된 [아이디] , [이메일] 입력 클릭 → [이메일로 인증받기]후 찾기가 가능합니다.




소셜(카카오)로 가입한 아이디를 사람인 가입 아이디로 전환하는 것은 불가능합니다




같은 사람의 계정이라도, 소셜 가입 계정과 사람인 가입 계정은 별도 아이디로 취급됩니다.


이력서, 스크랩 등 개인활동 역시 개별적으로 저장됩니다.



소셜 계정은 사람인 내에서는 비밀번호 변경이 불가능합니다.



따라서 해당 소셜 플랫폼 사이트(카카오)에 접속하신 후 비밀번호를 변경하셔야 합니다.


변경 후에는 사람인 사이트에서 소셜 로그인 버튼을 눌러, 변경하신 비밀번호를 입력하시면 됩니다.



참고로, 사람인 사이트에서 임시 비밀번호를 발급받은 소셜 계정의 경우에도 비밀번호 변경은 불가능합니다.


서비스 이용


[마이펫] → [진료내역]에 들어가시면 지난 진료기록을 모두 확인하실 수 있습니다.


보다 자세한 진료내역을 원하실 경우 [진료내역] 목록 중 원하시는 내역을 클릭하시면 확인하실 수 있습니다.


☞진료내역 바로가기(클릭)


네 맞습니다. 저희 업체와 연계된 병원에서 이용이 가능하십니다.


이용 가능한 병원은 [병원찾기]에 등록된 병원을 확인 부탁드립니다.

현재는 이용 가능한 병원이 많지 않지만 앞으로 서비스를 더 확장해 나갈것입니다.




수정 가능합니다.




1. [마이펫] → [펫정보] 클릭 → 수정을 원하는 강아지 리스트 클릭




2. [수정]버튼 클릭




3. 원하는 정보 수정 후 [작성]버튼 클릭



아닙니다.일반회원은 진료차트에 대해 작성은 불가능합니다.


작성은 담당의만 작성이 가능하며 일반회원은 진료차트 보는것만 가능합니다.



또한 저장된 진료차트는 다른병원과 공유된다는 점 확인바랍니다.


카드


카드발급은 홈페이지에서 가능합니다.


카드 서비스 이용을 원하시는 분은 회원가입 후 이용이 가능합니다.



로그인 후 메인페이지에서 [펫카드] → [카드발급] 클릭 후 발급신청이 가능합니다.



발급을 원하시는 분은 카드 발급 안내문을 꼭 읽어보신 후 신청 바랍니다.



☞카드발급 바로가기(클릭)

기업 회원가입/로그인


기업회원 가입은 회원가입 페이지에서 진행 가능하며, 회원가입은 무료입니다.



일반회원 가입과 양식이 다르니 꼭 확인 후 작성 바랍니다.



기업회원 가입과 동시에 진료차트 작성, 예약관리를 이용하실 수 있습니다.



1. 메인페이지 우측에 [회원가입]버튼 클릭



2. 상단에 [기업회원]탭 클릭


3. 회원가입 정보 기입 후 [가입]버튼 클릭


☞회원가입 바로가기(클릭)


로그인 영역의 아이디/비밀번호 찾기'를 통해 알 수 있습니다.




아이디 찾기 : 등록된 [이메일] , [연락처] 입력 → [아이디 찾기] 버튼 클릭




비밀번호 찾기 : 등록된 [아이디] , [이메일] 입 클릭[이메일로 인증받기]후 찾기가 가능합니다.
